# ルールの実装方針

本シミュレーターは下記の競技団体およびネット麻雀の公式ルールに対応する予定です。
そのため、それらで採用されていないルールの内、実装が複雑になるルールについては現在のところ実装する予定はありません (実装が単純なら対応することもあります)。  
対象の競技規定および公式ルールが変更された際は対応する予定です。

## ルール実装対象の競技団体 (順不同)

- 最高位戦日本プロ麻雀協会
- 日本プロ麻雀連盟
- 麻将連合-μ-
- 日本プロ麻雀協会
- RMU
- Mリーグ

## ルール実装対象のネット麻雀 (順不同)

- 天鳳 (Tenhou)
- 雀魂 (Mahjong Soul)
- 麻雀一番街 (Riichi City)
- 龍龍 (RonRon)
